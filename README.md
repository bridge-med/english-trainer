# Daily English

瞬間英作文トレーナー。通勤の5分で、日本語を見て英語を思い浮かべる練習。

🌐 **Live**: https://bridge-med.github.io/english-trainer/

## できること

- **単語 220個** / **フレーズ 135個** の日本語↔英語学習
- **4モード**: リビール / 4択 / 並べ替え / ディクテーション
- **音声読み上げ** (ブラウザ標準API、外部API不要)
- **レベル別 × カテゴリ別 複数選択** フィルタ
- **SRS間隔反復** と **復習モード** で弱点集中学習
- **お気に入り** と **カスタム追加** で自分専用リストに
- **連続学習日数・習得率・カレンダー** で進捗可視化

## レベル構成

**汎用 (Lv1-6)** 日常英会話
- Lv1 あいづち / Lv2 体調・気持ち / Lv3 食事・買い物
- Lv4 移動・道案内 / Lv5 仕事の基本会話 / Lv6 雑談

**BRIDGE (Lv7-9)** 医療経営コンサル特化
- Lv7 医療現場 基礎（患者対応・診察）
- Lv8 病院運営・KPI（改善率・稼働・ボトルネック）
- Lv9 コンサル定番（align・baseline・scope）

## 使い方

https://bridge-med.github.io/english-trainer/ をブラウザで開くだけ。

データはブラウザ内(localStorage)に保存。サーバー送信なし、ログイン不要。同じブラウザなら次回も続きから。

## 開発

単一HTMLファイル(`index.html`)。フレームワークなし、依存はGoogle Fontsのみ。
